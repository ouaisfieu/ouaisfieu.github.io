# Images à produire pour Alertes Sociales Belgique

## Open Graph Images (1200×630 px)

Toutes les images OG doivent suivre la même esthétique :
- **Fond** : Dégradé sombre (#1a1a1a → #2d2d2d)
- **Typographie** : Playfair Display pour le titre principal
- **Couleur accent** : Rouge alerte (#b91c1c)
- **Logo** : Point d'exclamation stylisé en haut à gauche

### Fichiers requis :

1. **og-home.png** (1200×630)
   - Titre : "Alertes Sociales Belgique"
   - Sous-titre : "Surveillance citoyenne des droits sociaux"
   - Chiffres clés en overlay : "184 463 | 270 | 2 300"

2. **og-demantelement.png** (1200×630)
   - Titre : "Le démantèlement orchestré"
   - Visual : Silhouettes de manifestants ou graphique descendant
   - Badge : "184 463 exclusions"

3. **og-depossession.png** (1200×630)
   - Titre : "La dépossession citoyenne"
   - Visual : Timeline 1980-2026
   - Badge : "40 ans de néolibéralisme"

4. **og-strategie.png** (1200×630)
   - Titre : "Stratégie de publication"
   - Visual : Logos médias (stylisés) ou réseau de connexions
   - Badge : "8+ canaux"

## Twitter Cards (même format que OG)

Copier les images OG ou créer des versions légèrement adaptées.

## Favicons

Utiliser https://realfavicongenerator.net/ avec le logo "!" :

- **favicon.ico** (multi-résolution)
- **favicon-16x16.png**
- **favicon-32x32.png**
- **apple-touch-icon.png** (180×180)
- **android-chrome-192x192.png**
- **android-chrome-512x512.png**
- **site.webmanifest**

### Spécifications du favicon :
- Fond : Rouge alerte (#b91c1c)
- Icône : Point d'exclamation blanc
- Forme : Carré avec coins légèrement arrondis

## Logo

1. **logo.png** (400×100)
   - Version horizontale : "! Alertes Sociales"
   - Fond transparent

2. **logo-square.png** (512×512)
   - Version carrée : "!" seul
   - Pour les réseaux sociaux

## Outils recommandés

- **Canva** : Templates Open Graph gratuits
- **Figma** : Design professionnel
- **GIMP/Photoshop** : Édition avancée
- **Squoosh** : Compression optimale (WebP, AVIF)

## Checklist SEO images

- [ ] Alt text descriptif sur toutes les images
- [ ] Compression optimale (<100 KB pour OG images)
- [ ] Noms de fichiers descriptifs (pas de espaces, lowercase)
- [ ] Format : PNG pour logos/graphiques, WebP pour photos
- [ ] Dimensions exactes respectées (évite le redimensionnement)
